# Budget-Tracker

## Description
Budget Tracker application using MongoDB, Node, and Express. It allows a user to add expenses and deposits to their budget with or without data/internet connection. They will then be able to enter transactions offline, which will be populated when brought back online.


## Links
* [Github Repository](https://github.com/jkaganovsky/Budget-Tracker)
* [Heroku](https://on-off-budget-tracking.herokuapp.com/)


## Installation & Usage
FEATURES
1. Adds a user's expenses and deposits.
1. Tracks a user's withdrawals and deposits without data/internet connection.
1. Views/Access a user's expenses anytime.
